## :office: 연구실 소개
: DEBA 연구실은 디지털경제의 표준이 되어가고 있는 `빅데이터와 머신/딥러닝이란 인공지능 이론을 활용`하여,     
`경제산업 분야의 문제`를 `데이터`를 기반으로 `정량적으로 분석하고 의사결정`함으로써 `전략적으로 문제를 해결`하기 위한 방법을 연구합니다.     
- **1기 최종멤버:** `손도언, 차명주, 장민재, 임보민`
